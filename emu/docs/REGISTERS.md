# Registers

## A

Accumulator register

Code: 000

## B

Code: 001

## C

Code: 010

## D

Code: 011

## E

Code: 100

## F

Flag register

Flags (and bit location, starting at lowest order bit):

- 0: Overflow
- 1: Negative

Code: 101

## HL

Address register

Code: 110

## SP

Stack pointer

## PC

Program counter
